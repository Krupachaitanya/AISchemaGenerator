What is this?

AI Schema Generator is a VS Code extension that helps you generate complete projects, auto-create config/files, and build APIs instantly.

1. One-click Schema Creation

Creates project structure automatically (Config/, Data/, ForFrontEnd/, V1/). Generates essential files like .env, app.js, and package.json.

2. API Builder

Converts your generated schema into working APIs. Provides a ready-to-run Node.js backend seamlessly.

3. Integrated Workflow

Everything happens inside VS Code. No manual setup, no boilerplate writing, and no manual folder creation required.
